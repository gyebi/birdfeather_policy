# adinkragame
 🪔 Gye Nyame Memory Game

Gye Nyame Memory Game is a browser-based memory matching game inspired by Adinkra symbols from Ghana, blending interactive gameplay with cultural education. Players flip cards to match traditional symbols, and each successful match reveals the symbol’s name and meaning, turning the game into a learning experience as well as a challenge of memory.

🎮 Features

Classic memory-matching gameplay

Uses authentic Adinkra symbols

Displays the name and cultural meaning of each symbol upon a successful match

Progressive discovery panel that builds as matches are found

Attempt counter and restart functionality

Clean, responsive design built with vanilla HTML, CSS, and JavaScript

🧠 Educational Focus

Beyond entertainment, the game introduces players to the symbolism and philosophy behind Adinkra symbols such as Gye Nyame, Sankofa, Dwennimmen, and Nsoromma, promoting cultural awareness and learning through play.

🛠️ Built With

HTML5

CSS3

JavaScript (Vanilla)

🚀 Purpose

This project was built as a learning exercise in DOM manipulation, game logic, and asset management, while also celebrating African heritage and visual storytelling.
